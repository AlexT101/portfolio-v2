# Portfolio

This is my portfolio website, showcasing my frontend projects for hackathons, clubs, and other interests.

## Technologies Used
- React
- TypeScript
- HTML
- CSS

## How to Use

### Clone project

```bash
git clone https://github.com/AlexT101/portfolio-v2
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```
